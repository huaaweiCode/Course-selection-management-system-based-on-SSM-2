# 基于SSM的选课管理系统2

#### Description
基于SSM的选课管理系统2拥有三种角色

- 管理员：菜单管理、角色权限管理、用户管理、课程管理、互动管理、成绩管理、公告管理、学生选课列表等
- 用户：登录注册、选课、查看自己所选课程和成绩、其它具体功能可以通过管理员进行设置
- 教师：登录注册、打分、查看自己的课程和学生、其它具体功能可以通过管理员进行设置
